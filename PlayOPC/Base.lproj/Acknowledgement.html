CocoaAsyncSocket

Asynchronous socket networking library for Mac and iOS. See also https://github.com/robbiehanson/CocoaAsyncSocket/

MBProgressHUD

MBProgressHUD is an iOS drop-in class that displays a translucent HUD with an indicator and/or labels while work is being done in a background thread. See also https://github.com/jdg/MBProgressHUD
